A showcase of created Unity Materials using shader graphs

Sujections for Textures to use on these shaders

Poly Haven — great free CC0 textures (https://polyhaven.com)

ambientCG — huge free PBR library

TextureCan — good free materials
ShareTextures — free PBR textures
