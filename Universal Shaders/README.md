## Shader Breakdown

This graph is built as a reusable URP master material setup, intended to expose common controls (color, surface response, and effect masks) in one place.

- Use it as a base graph for quickly creating stylized variants.
- The effect is designed for parameter-driven iteration rather than hardcoded looks.
- Pair it with per-material overrides to produce different results from the same shader graph.
